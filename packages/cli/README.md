# @mcpshield/cli

Command-line interface for [MCPShield](../../README.md).

## Install

```bash
npm install -g @mcpshield/cli
```

Or run without installing:

```bash
npx @mcpshield/cli init
```

## Commands

```
mcpshield init                  Scaffold mcpshield.yaml
mcpshield start [--config <p>]  Start the gateway
mcpshield policy lint           Validate policies in your config
mcpshield pack list             List installed + active packs
mcpshield pack add <name>       Install a pack from npm
```
